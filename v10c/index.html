How to read this site

Three tiers of evidence. Every claim on every page carries a badge: hard data for measured findings with a named report and N; triangulated for findings supported by multiple independent sources or external benchmarks; hypothesis for in-validation findings we have named the evaluation for. Claims with no source are dropped, not soft-tagged.

Scope discipline. Every number is attached to the program it was measured in. "In CHC, cost per verified visit fell 22%" is a CHC finding, not a Connect-wide claim. The mechanism is platform-portable; the number is not.

Open questions on every depth page. Each depth page closes with one to three unresolved tensions, sourced from caveats in insights.md and nuggets.md. We don't fabricate open questions and we don't omit them. Honesty about what we don't know is the rate-limiter on the methodology, not the format.
