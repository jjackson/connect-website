How to read the dispatches.

Three conventions before you click in. They are the same conventions we apply when we write up a program for a funder, and they are the reason this site looks more like a notebook than a brochure.

Scope discipline. Every number is attached to the program it was measured in. "In CHC, cost per verified visit fell 22%" is a CHC finding, not a Connect-wide claim. Where a mechanism is portable across programs we say so; where the number is not yet replicated we say that too.

Progression of thinking. Each LDVP page (Learn / Deliver / Verify / Pay) is laid out as a timeline. We had a question. We tried something. The data came back. We changed the design. The progression is the interesting part — the final answer in isolation is usually less informative than the path that produced it.

Open questions. Every depth page closes with one to three unresolved tensions. We publish the gaps because the alternative — claiming we have a methodology when we have a marketing site — is the failure mode that funder due-diligence is designed to catch.
